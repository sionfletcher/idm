  • Sublime Text - My text editor of choice if you’d rather work offline
  • Google Chrome - My browser of choice for development, has the best tools in my opinion. If you’re using it right now, Click on View → Developer → Developer Tools to see what I mean.

References and Help

W3 Schools - the go to reference for all things web development. You can find detailed explainations of the code you will be using throughout this course, and very useful examples of how they work. I’d say when working on web projects, I use this site every other day — but if you’re starting out, it should be the first thing you open up before you get to work.

Stack Overflow - an incredible resource. It’s a question and answer forum for programmers, where some charitable and extremely talented programmers spend hours of their time, freely answering people’s questions for kudos. I use this website constantly when I program my own apps and games (and I mean hundreds of visits a day). Even so, I’ve only ever had to post two questions on there, as someone invarialbly posed the same question before me. Here’s one of my questions if you want an example of the caliber of feedbck you can get. If you do want to ask a question, make absolutely sure it has not been answered before, so you can keep your own kudos high!

Further Learning

Khan Academy is a site where you can take a course in many subjects, ranging from Micro Economics to the Arts and Humanities. For us, they have an excellent course on computer programming which will give you a great grounding if you really want to get into this stuff.

Code School - similarly to Khan Academy, you can take online courses here, although some are paid, the introductions are free, and well worth a look

Apress Books - if you can’t find what you need in the library, I’ve personally found the Apress books a good start when I want to learn a new language. It might seem silly to buy programming books, with the languages developing so quickly, and the resources available online; however, a book is the best place to start. When you start learning, it’s difficult to even know what to search for online. You will invarably land on a page that is out of your depth, and stumble accross people arguing on what solution is best, for problems you can’t even understand yet.

Reading textbooks first is a wise move. It takes the frustration and confusion out of learning, and a good author will never discuss a problem that contains parts which haven’t been covered previously. They also make great bedtime reading, as you’ll be out in minutes.
